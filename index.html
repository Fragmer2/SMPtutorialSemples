▸ Video Sequence · commands to copy

step by step — Papi setup, world, regions, NPCs, holograms, permissions (from the video)
1. PlaceholderAPI — download extensions 
/papi ecloud download Vault
/papi ecloud download LuckPerms
/papi ecloud download Player
/papi reload
2. Void world (example) and spawn 
/world create example:void type flat the-void
/setspawn
/worldborder set 50
3. WorldEdit — select and fill with stone 
//wand
//set 1
✦ 4. Holograms (HolographicDisplays) 
/hologram create text Welcome
/hologram edit Welcome addline Hello write /rtp to start
/hologram edit Welcome removeline 1
5. NPC — Seller / Buyer (Citizens) 
/npc create Seller
/npc displayname Seller &6&lBuyer
/npc action Seller any_click add player_command buyer
6. Region protection (WorldGuard) 
//expand vert
/region claim Spawn
# Then configure flags (flag examples can be taken from the video)
7. Second NPC — Auction 
/npc copy Seller Auction
/npc displayname Auction &6&lAuction
/npc action Auction any_click add player_command ah
8. WorldBorder for worlds 
/worldborder set 10000   # for the main world
# in Nether similarly: /worldborder set 10000 (while in the desired world)
# in End: /worldborder set 50000
9. LuckPerms — add rtp permissions in Nether/End 
/lp editor
# open the link in your browser → select the default group
# find EZRTP permissions for world_nether / world_the_end
# add: ezrtp.bypass.world_nether, ezrtp.bypass.world_the_end
# save and apply
after all commands: don't forget to deop, check /rtp, /ah, /buyer, /guide. All plugin configs are already inserted above.
